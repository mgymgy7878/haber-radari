# Antigravity Rules

## EKRAN GÖRÜNTÜSÜ ARŞİV TALİMATI
Her cihaz smoke veya görsel test turunda alınan ekran görüntülerini iki yere kaydet:

1. Repo evidence:
`evidence/android/<ilgili_test_adi>.png`

2. Kullanıcının masaüstü arşivi:
`Desktop\haber uygulaması ekran görüntüleri\<tur_no>\`

Örnek:
`Desktop\haber uygulaması ekran görüntüleri\tur_2\android_live_ui_news_value_v0_fixed.png`

Her yeni smoke/test turu için yeni klasör aç:
tur_1, tur_2, tur_3 ...

Bu klasör repo dışıdır; sadece kullanıcının görsel arşivi içindir.
PR evidence için repo-relative path kullanılmalıdır.
