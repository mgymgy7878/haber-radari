### Provider interface

```ts
interface SmartDigestProvider {
  readonly name: 'mock' | 'external';
  generate(input: SmartDigestInput, signal?: AbortSignal): Promise<SmartDigestProviderResult>;
}
```

Implementasyonlar: `MockSmartDigestProvider`, `ExternalSmartDigestProvider` (injectable `fetchFn`).

### External provider default açık mı?

**Kapalı.** Default env:

- `LLM_DIGEST_ENABLED=0`
- `LLM_DIGEST_PROVIDER=mock`
- `LLM_DIGEST_EXTERNAL_ENABLED=0`

Gerçek external çağrı için **her iki** flag gerekir: `LLM_DIGEST_ENABLED=1` + `LLM_DIGEST_PROVIDER=external` + `LLM_DIGEST_EXTERNAL_ENABLED=1` + geçerli `LLM_DIGEST_API_KEY`.

### Gerçek external çağrı yapıldı mı?

**Hayır (production).** Evidence ve testlerde injectable mock `fetch` kullanıldı (1 simüle çağrı — cache-hit senaryosu). Production API key veya gerçek OpenAI endpoint'ine istek gönderilmedi.

### Cache zorunluluğu

- `LLM_DIGEST_REQUIRE_CACHE=1` (default): her yazım sonrası `cache.get()` ile doğrulama; başarısızsa `CACHE_WRITE_FAILED`.
- Cache key: SHA256(normalized metadata input + promptVersion + digestVersion).
- Dosya: `apps/api/.cache/smart-digest/{cacheKey}.json`.
- Cache hit → external/mock provider çağrılmaz.

### Budget guard

- Dosya: `apps/api/.cache/smart-digest-budget/YYYY-MM-DD.json`
- Alanlar: `date`, `externalCalls`, `cacheHits`, `failures`, `budgetDenied`
- `LLM_DIGEST_DAILY_LIMIT` (default 20) aşılınca → external call yok, `FAILED` + `BUDGET_EXCEEDED`
- `LLM_DIGEST_PER_REQUEST_LIMIT` (default 3) aşılınca → kalan item'lar `PER_REQUEST_LIMIT`
- Endpoint HTTP 200 dönmeye devam eder

### API key / log güvenliği

- API key loglanmaz; `sanitizeProviderError()` key ve `Bearer` token redact eder.
- `logProviderEvent()` yalnızca: `cacheKey`, `provider`, `status`, `elapsedMs`, `errorCode`.
- Full input payload ve prompt/response default loglanmaz.
- Test: `smart-digest-service.test.ts` — log buffer'da secret key aranmaz.

### Full text kullanılmadı — kanıt

- `FORBIDDEN_DIGEST_FIELDS`: fullText, body, contentHtml, articleText, htmlContent, rawContent, scrapedText, rawHtml
- `normalizeSmartDigestInput()` forbidden alan varsa throw.
- External prompt yalnızca title, shortDescription, source metadata.
- Smart Feed: digest yalnızca `PUBLISH_MAIN` item'larda; `latestRssPreview`, `watchlistPreview`, `rawPreview`, `noisePreview` digest almaz.
