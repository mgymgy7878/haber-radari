**Date:** 2026-06-28

## Özet

Smart Feed response'a cache'lenebilir, metadata-only `smartDigest` alanı eklendi. Gerçek LLM provider çağrısı **yapılmadı** — mock/deterministic provider + dosya cache altyapısı kuruldu.

---

## SmartDigest Contract

```ts
smartDigest?: {
  status: "DISABLED" | "MOCKED" | "CACHED" | "GENERATED" | "FAILED";
  summary: string | null;
  keyPoints: string[];
  whyItMatters: string | null;
  confidence: "LOW" | "MEDIUM" | "HIGH";
  sourcePolicy: "METADATA_ONLY";
  modelProvider: "mock" | "disabled" | "external";
  cacheKey: string;
  generatedAt: string | null;
  errorCode?: string;
}
```

Top-level `smartDigestStats` eklendi: requested, cacheHits, cacheMisses, disabled, mocked, cached, generated, failed.

---

## Cache Key Algoritması

```
normalizedInput = normalizeSmartDigestInput(input)
  → kaynak sırası deterministik (sourceName + publishedAt)
  → boş/null trim
  → forbidden fields (fullText, body, contentHtml, articleText) reddedilir

cacheKey = SHA256(JSON.stringify({
  promptVersion,
  digestVersion,
  input: normalizedInput
}))
```

- `promptVersion`: env `LLM_DIGEST_PROMPT_VERSION` (default `v0.6.0`)
- `digestVersion`: env `LLM_DIGEST_VERSION` (default `v0.6.0`)

---

## Cache Storage

- **Path:** `apps/api/.cache/smart-digest/{cacheKey}.json`
- **Override:** `LLM_DIGEST_CACHE_DIR`
- **TTL:** `LLM_DIGEST_CACHE_TTL_MS` (default 24h)
- `.gitignore`: `apps/api/.cache/`

## Digest Input Alanları (metadata-only)

- clusterId, title, shortDescription
- category, publishDecision, publishReason
- sourceCount, sourceNames, publishedAt
- sources[]: sourceName, title, shortDescription, originalUrl, publishedAt
- importance, evidenceStatus, contentType, topicQuality (opsiyonel)

**Kullanılmayan / yasak:** fullText, body, contentHtml, articleText, scraping

---

## Pipeline Kuralları

- Digest yalnızca `PUBLISH_MAIN` item'lar için üretilir
- `watchlistPreview`, `rawPreview`, `noisePreview`, `latestRssPreview` → digest **yok**
- Digest hatası endpoint'i düşürmez → item.smartDigest.status=`FAILED`, HTTP 200
