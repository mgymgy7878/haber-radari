## Özet

v0.6.3, gerçek external LLM pilotu için operatör onay kapısı, genişletilmiş budget evidence ve output guard altyapısını ekler. Tüm kanıtlar injectable fake `fetch` ile üretildi.

## Approval gate

External digest çağrısı şu sırayla kontrol edilir:

1. `LLM_DIGEST_EXTERNAL_ENABLED=1`
2. `PUBLISH_MAIN` + eligibility (`uniqueSourceCount >= 2` veya yüksek önem)
3. `LLM_DIGEST_REQUIRE_OPERATOR_APPROVAL=1` ise `OPERATOR_APPROVED_REAL_LLM_DIGEST=1` (veya config override)
4. `LLM_DIGEST_API_KEY` dolu
5. Günlük / request başına budget

Onay yoksa: `status=FAILED`, `errorCode=OPERATOR_APPROVAL_REQUIRED`, HTTP 200 korunur, provider çağrılmaz.

## Budget guard

- Günlük limit varsayılan: **5** (`LLM_DIGEST_DAILY_LIMIT`)
- Request başına limit varsayılan: **1** (`LLM_DIGEST_PER_REQUEST_LIMIT`)
- Evidence dosyası: `apps/api/.cache/smart-digest-budget/YYYY-MM-DD.json`
- Alanlar: `date`, `externalCalls`, `cacheHits`, `failures`, `budgetDenied`, `approvalDenied`, `lastUpdatedAt`
